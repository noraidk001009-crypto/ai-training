# Dice Roller

### here are the rules for how to play mmain.py!!!

## How to Run

1. Make sure Python 3 is installed.
2. Open a terminal in this `assignment` folder.
3. Run:

```powershell
python main.py
```

4. Enter how many dice to roll when prompted (press Enter for 1 die).
