## What is a bug?
A **bug** is a mistake in code that causes a program to crash or behave incorrectly.

For example, this Python code is correct because the text is inside quotation marks: print("handle is a pig")
But this is wrong because the quotation marks are missing: print(handle is a pig)
That kind of mistake is a **bug**.
